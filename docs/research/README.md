# Research notes

These reports preserve dated evidence used by architecture decisions. They do
not replace the current package map or an accepted decision. Start with the
[workspace architecture](../workspace-architecture.md) for the current state
and the [architecture decision log](../decisions/README.md) for governing
constraints.

## Available reports

- [Architecture research ledger](architecture-ledger.md): source provenance,
  limitations, and Adopt, Adapt, Defer, or Reject classifications from the
  architecture-cohesion campaign.
- [Rust measurement gate](rust/measurement-gate.md): the 2026-07-18 performance
  measurements and revisit criteria supporting ADR 0004.

Treat every report as a dated snapshot. Re-run measurements and recheck external
sources when a new decision depends on current behavior.
